You are a professional UI designer. Design the Khatabook screen for Diya Wholesaler Web App.

Screen Number: DIYA-WS-06  
Screen Name: Khatabook Home

Objective:
Display all retailer dues in a simple and actionable layout for faster payment recovery.

UX Components:
- Top Summary Tiles (3):
  1. Total Outstanding Due
  2. Overdue Amount
  3. Payments Received this Month
- Filters:
  - All | Pending | Overdue | Partial Paid | Settled
- Retailer Dues List:
  - Retailer Name, Due Amount, Overdue Amount, Last Payment Date
  - Status Colors: Green (Clear), Orange (Pending), Red (Overdue)
  - CTA: “Request Payment” button
- Search Bar (Retailer name)

Interactions:
- Click retailer → Retailer Ledger Screen
- Filter pills update list live
- Hover reveal: call, WhatsApp icons

Data Attributes:
- outstanding_due
- overdue_due
- last_payment_date
- payment_delay_rank

Animations:
- Tile count animation
- Pulse effect on critical overdue red badges
- Smooth fade for filter changes

Style Reference:
Clean, minimal, 21stdev inspiration, big numbers, bold colors.
